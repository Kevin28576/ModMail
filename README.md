# 什麼是 ModMail Bot ?
Modmail 是一個在功能和用途上都類似於 Reddit 的 Modmail。它作為共享收件箱，供群組工作人員以無縫方式與用戶交流。
    
# Discord.js V14 ModMail Bot - 介紹:
ModMail Bot 是一個當用戶想要聯絡管理員時可以直接私訊機器人來做到與管理員聯絡的功能，此機器人資料庫使用`Quick.db`的強大功能，此機器人也支持斜線指令處理。<br>
